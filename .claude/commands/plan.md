# plan

Execute the adaptive-checklist-planner agent with META Framework principles.

## Usage

```
/plan <task-description>
```

## Description

Generates comprehensive, architecturally-aligned checklist plans with:

- **META Coordinates** (X/Y/Z/W axis calculations)
- **Ripple Chain Analysis** (6 dimensions: registry, pools, entities, database, dashboard, validators)
- **3D Validation Graphs** (sequential, lateral, diagonal dependencies)
- **Hierarchical Numbering** (Phase N → Task N.N → Subtask N.N.N)
- **Evidence-Based Discovery** (skeptical context acquisition with INVESTIGATE→ACTION pattern)
- **Adaptive Folder Maintenance** (updates EVOLUTION-LOG.json and META-ARCHITECTURE-DIAGRAM.md)

## Inherited Skepticism

This agent embodies verification discipline:

- Tool calibration before execution
- All claims verified against empirical evidence
- No assumptions without Glob/Grep/Read confirmation
- Verification rate tracking (must be >= 0.8)
- Evidence chain documentation

## META Framework Integration

The agent maintains the adaptive folder structure:

- `.claude/workspace/_meta-templates/adaptive/EVOLUTION-LOG.json` - Pattern learning and promotion
- `.claude/workspace/_meta-templates/adaptive/META-ARCHITECTURE-DIAGRAM.md` - Architecture visualization
- `.claude/workspace/_meta-templates/adaptive/*-checklist.md` - Generated checklists

## Pattern Learning

After checklist execution, provide feedback to enable learning:

```
Report execution feedback for checklist {id}:
- Discoveries used: ["base-tracker.ts", "intent-manager.ts"]
- Discoveries that prevented issues: ["base-tracker.ts"]
- New anti-patterns: ["mega-class"]
- Success rating: 9/10
```

The system will:
1. Track discovery value
2. Promote patterns crossing threshold (>= 5 uses)
3. Update discovery heuristics
4. Propose template evolution when compression ratio >= 1.5

## Architectural Compliance

Generated checklists enforce:

- **7 Laws** (Forward-Only, No Callbacks, Inheritance, Security-Base, Auth-State, Human-Authority, Registry)
- **AVOID→ENFORCE Mapping** (no shortcuts, fallbacks, dual-paths, for-now, marketing superlatives)
- **DOM Factory** (DOM.createElement with entity parameter, no document.createElement)
- **Engine.logger** (schema-only API with required fields: component, execution)
- **CSS Tokens** (var(--*), no px/hex hardcoding)
- **Registry Pattern** (Engine.registry.resolve, no direct imports)
- **Validator Coverage** (architectural patterns have detector enforcement)

## Safe Iteration Loops

The agent inherits safe iteration patterns:
- `max_iterations` limit (default: 10)
- `iteration_count` tracking
- `goal_achieved` termination condition
- No unbounded loops (WHILE true forbidden)

## Example

```
/plan Implement IntentFlowTracker for LAW 2 compliance
```

Generates:
- `.claude/workspace/_meta-templates/adaptive/intent-flow-tracker-checklist.md`
- Updates EVOLUTION-LOG.json with checklist metadata
- Updates META-ARCHITECTURE-DIAGRAM.md if new patterns emerge
- Calculates META coordinates (X/Y/Z/W)
- Analyzes ripple chains across 6 dimensions
- Generates 3D validation graph
- Decomposes into hierarchical phases
